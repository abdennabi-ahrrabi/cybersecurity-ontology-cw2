# Cybersecurity Incident Response Ontology

MSc Computer Science - COM759 Knowledge Engineering
Ulster University

## Author
Abdennabi Ahrrabi

## Description
OWL ontology for healthcare ransomware incident response, 
built in Protégé 5.6.

## Features
- 49 classes, 23 object properties, 10 datatype properties
- Property chain for automatic attack path inference
- 7 defined classes for automated reasoning
- MedCare Hospital ransomware scenario (54 individuals)

## How to Use
1. Download the .owl file
2. Open in Protégé 5.6+
3. Run HermiT reasoner to see inferences
